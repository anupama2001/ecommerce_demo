# Flutter E-Commerce Demo App

## Features
- Authentication (Dummy with SharedPreferences)
- Product Listing with Search & Category Filter
- Cart Management with Quantity Control
- Order System with Date & Status

## Architecture
- MVVM-inspired structure
- Provider for State Management
- Modular folder structure
- Clean separation of logic and UI

## Folder Structure
lib/
 ├── models/
 ├── providers/
 ├── screens/
 ├── widgets/
 └── data/

## How to Run
flutter pub get  
flutter run  

## APK
Release APK included in submission.
